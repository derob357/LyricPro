# Payment Page + Avatar Locker Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a tier/rank Status header to the existing `/shop` page and ship a new `/avatars` Avatar Locker that lets users unlock equipped avatars by spending Golden Notes.

**Architecture:** New `avatars` and `user_avatars` tables + a `users.equipped_avatar_id` column. New `avatars` tRPC router with `list / unlock / equip`. New `<UserAvatar />` shared component renders the equipped avatar across PersistentHeader, Profile, Leaderboards, Gameplay, FinalResults. Stripe wiring is reused as-is from the v0.3.0 Golden Notes work; this plan only consumes it.

**Tech Stack:** Drizzle (Postgres), tRPC, React + Vite, Wouter (routing), Tailwind, vitest.

Create `server/routers/avatars.ts`:

```ts
import { z } from "zod";
import { and, asc, eq, sql } from "drizzle-orm";
import { TRPCError } from "@trpc/server";
import { router, protectedProcedure } from "../_core/trpc";
import { rateLimit } from "../_core/rateLimit";
import { getDb } from "../db";
import {
  avatars,
  userAvatars,
  users,
  goldenNoteBalances,
  goldenNoteTransactions,
} from "../../drizzle/schema";

export const avatarsRouter = router({
  // Catalog + ownership state for the signed-in user. The page renders an
  // "Owned" grid and an "Available to unlock" grid from the same response.
  list: protectedProcedure.query(async ({ ctx }) => {
    const db = await getDb();
    if (!db) {
      throw new TRPCError({
        code: "INTERNAL_SERVER_ERROR",
        message: "Database unavailable",
      });
    }

    const ownedRows = await db
      .select({ avatarId: userAvatars.avatarId })
      .from(userAvatars)
      .where(eq(userAvatars.userId, ctx.user.id));
    const ownedIds = new Set(ownedRows.map((r) => r.avatarId));

    const catalogRows = await db
      .select()
      .from(avatars)
      .where(eq(avatars.isActive, true))
      .orderBy(asc(avatars.sortOrder));

    const [me] = await db
      .select({ equippedAvatarId: users.equippedAvatarId })
      .from(users)
      .where(eq(users.id, ctx.user.id));

    return {
      catalog: catalogRows.map((a) => ({
        id: a.id,
        slug: a.slug,
        name: a.name,
        imageUrl: a.imageUrl,
        rarity: a.rarity,
        priceGn: a.priceGn,
        owned: ownedIds.has(a.id),
      })),
      equippedAvatarId: me?.equippedAvatarId ?? null,
    };
  }),
});
```

Create the in-memory DB mock helper in `server/avatars.test.helpers.ts`:

```ts
// Minimal in-memory fake of the Drizzle DB surface used by avatars.ts.
// Only the operators avatars.ts actually uses are implemented. Keeps tests
// fast and side-effect-free; integration coverage runs against real Postgres
// in the manual smoke at the end of the plan.

type AvatarRow = {
  id: number;
  slug: string;
  name: string;
  imageUrl: string;
  rarity: "starter" | "common" | "rare" | "epic" | "legendary";
  priceGn: number;
  isActive: boolean;
  sortOrder: number;
  createdAt?: Date;
};
type UserAvatarRow = {
  userId: number;
  avatarId: number;
  acquiredAt?: Date;
  acquiredVia: "starter" | "purchase" | "admin_grant";
  spentGn: number;
};
type UserRow = {
  id: number;
  equippedAvatarId: number | null;
};
type BalanceRow = {
  userId: number;
  balance: number;
  lifetimePurchased: number;
  lifetimeSpent: number;
};

const state = {
  avatars: [] as AvatarRow[],
  userAvatars: [] as UserAvatarRow[],
  users: new Map<number, UserRow>(),
  balances: new Map<number, BalanceRow>(),
  txLog: [] as Array<{ userId: number; amount: number; kind: string; reason: string | null; balanceAfter: number }>,
};

export function resetMockDb() {
  state.avatars = [];
  state.userAvatars = [];
  state.users.clear();
  state.balances.clear();
  state.txLog = [];
}
export function seedAvatars(rows: AvatarRow[]) { state.avatars.push(...rows); }
export function seedUserAvatar(userId: number, avatarId: number, via: UserAvatarRow["acquiredVia"], spentGn: number) {
  state.userAvatars.push({ userId, avatarId, acquiredVia: via, spentGn });
}
export function setMockUser(id: number, row: Partial<UserRow>) {
  state.users.set(id, { id, equippedAvatarId: null, ...row, id });
}
export function setMockBalance(userId: number, balance: number) {
  state.balances.set(userId, { userId, balance, lifetimePurchased: 0, lifetimeSpent: 0 });
}
export function getMockBalance(userId: number): number {
  return state.balances.get(userId)?.balance ?? 0;
}
export function getTxLog() { return state.txLog.slice(); }
export function getOwned(userId: number): number[] {
  return state.userAvatars.filter((r) => r.userId === userId).map((r) => r.avatarId);
}
export function getEquipped(userId: number): number | null {
  return state.users.get(userId)?.equippedAvatarId ?? null;
}

// Drizzle-shaped fake. Only the call-chains used by the router are honored.
export async function getDb() {
  return {
    select: (cols?: unknown) => ({
      from: (table: { _: { name: string } } | unknown) => {
        const tableName = (table as { _?: { name?: string } })?._?.name
          ?? inferTableName(table);
        const builder = {
          _table: tableName,
          _where: undefined as ((row: Record<string, unknown>) => boolean) | undefined,
          where(predFromDrizzle: (row: Record<string, unknown>) => boolean) {
            this._where = predFromDrizzle;
            return this;
          },
          orderBy() { return this; },
          limit(n: number) { return this._run(n); },
          then(onFulfilled: (rows: Record<string, unknown>[]) => unknown) {
            return Promise.resolve(this._run()).then(onFulfilled);
          },
          _run(limit?: number): Record<string, unknown>[] {
            let rows: Record<string, unknown>[] = [];
            if (this._table === "avatars") rows = state.avatars as unknown as Record<string, unknown>[];
            else if (this._table === "user_avatars") rows = state.userAvatars as unknown as Record<string, unknown>[];
            else if (this._table === "users") rows = Array.from(state.users.values()) as unknown as Record<string, unknown>[];
            else if (this._table === "golden_note_balances") rows = Array.from(state.balances.values()) as unknown as Record<string, unknown>[];
            if (this._where) rows = rows.filter(this._where);
            return limit ? rows.slice(0, limit) : rows;
          },
        };
        return builder;
      },
    }),
    insert: (table: unknown) => ({
      values(row: Record<string, unknown> | Record<string, unknown>[]) {
        const list = Array.isArray(row) ? row : [row];
        const tableName = inferTableName(table);
        for (const r of list) {
          if (tableName === "user_avatars") state.userAvatars.push(r as UserAvatarRow);
          else if (tableName === "golden_note_transactions") state.txLog.push(r as typeof state.txLog[number]);
        }
        return { onConflictDoNothing() { return Promise.resolve(); } };
      },
    }),
    update: (table: unknown) => ({
      set(patch: Record<string, unknown>) {
        return {
          where(predicate: (row: Record<string, unknown>) => boolean) {
            const tableName = inferTableName(table);
            return {
              returning() {
                if (tableName === "users") {
                  for (const u of state.users.values()) {
                    if (predicate(u as unknown as Record<string, unknown>)) {
                      Object.assign(u, patch);
                    }
                  }
                  return Promise.resolve([]);
                }
                if (tableName === "golden_note_balances") {
                  const matching: BalanceRow[] = [];
                  for (const b of state.balances.values()) {
                    if (predicate(b as unknown as Record<string, unknown>)) {
                      Object.assign(b, applyPatch(b, patch));
                      matching.push(b);
                    }
                  }
                  return Promise.resolve(matching.map((b) => ({ newBalance: b.balance })));
                }
                return Promise.resolve([]);
              },
              then(onFulfilled: (v: unknown) => unknown) {
                return Promise.resolve(this.returning()).then(onFulfilled);
              },
            };
          },
        };
      },
    }),
    transaction: async (fn: (tx: unknown) => Promise<unknown>) => {
      // Trivial passthrough — the fake is single-threaded so atomicity is
      // implicit. Real concurrency is verified by the integration smoke.
      return fn(await getDb());
    },
  };
}

function inferTableName(t: unknown): string {
  // The drizzle table objects expose a Symbol(table_name) but the simplest
  // way to identify them in tests is by the imported reference. We rely on
  // the helpers passing the imported object through; if that fails, the
  // string name lives on `_.name` after Drizzle wraps it.
  const obj = t as { _?: { name?: string }; tableName?: string };
  return obj?._?.name ?? obj?.tableName ?? "";
}

function applyPatch(row: Record<string, unknown>, patch: Record<string, unknown>) {
  const out: Record<string, unknown> = {};
  for (const [k, v] of Object.entries(patch)) {
    if (typeof v === "function") {
      // Drizzle SQL fragments arrive as objects/closures here; the test
      // pre-evaluates simple `balance - cost` and `lifetimeSpent + cost`
      // shapes via the calling code — keep the fake honest by not
      // attempting to evaluate them.
      out[k] = v;
    } else {
      out[k] = v;
    }
  }
  return out;
}
```

```ts
  // Spend Golden Notes to unlock an avatar. Server-owned price; race-safe
  // debit using UPDATE ... WHERE balance >= cost RETURNING — same pattern as
  // goldenNotes.spend so concurrent unlocks can't double-debit. On success,
  // auto-equips the new avatar.
  unlock: protectedProcedure
    .input(z.object({ avatarId: z.number().int().positive() }))
    .mutation(async ({ ctx, input }) => {
      rateLimit("avatars.unlock", ctx.user.id, { max: 30, windowMs: 60_000 });
      const db = await getDb();
      if (!db) throw new TRPCError({ code: "INTERNAL_SERVER_ERROR", message: "Database unavailable" });

      // 1. Look up avatar (server-owned price)
      const [avatar] = await db
        .select()
        .from(avatars)
        .where(eq(avatars.id, input.avatarId))
        .limit(1);
      if (!avatar) throw new TRPCError({ code: "NOT_FOUND", message: "Avatar not found" });
      if (!avatar.isActive) {
        throw new TRPCError({ code: "BAD_REQUEST", message: "This avatar is no longer available" });
      }

      // 2. Reject double-purchase
      const existing = await db
        .select({ avatarId: userAvatars.avatarId })
        .from(userAvatars)
        .where(and(eq(userAvatars.userId, ctx.user.id), eq(userAvatars.avatarId, input.avatarId)))
        .limit(1);
      if (existing.length > 0) {
        throw new TRPCError({ code: "BAD_REQUEST", message: "You already own this avatar" });
      }

      const cost = avatar.priceGn;

      // 3-6. Race-safe debit + ownership row + audit log + auto-equip
      return await db.transaction(async (tx) => {
        const debited = await tx
          .update(goldenNoteBalances)
          .set({
            balance: sql`${goldenNoteBalances.balance} - ${cost}`,
            lifetimeSpent: sql`${goldenNoteBalances.lifetimeSpent} + ${cost}`,
            updatedAt: new Date(),
          })
          .where(
            and(
              eq(goldenNoteBalances.userId, ctx.user.id),
              sql`${goldenNoteBalances.balance} >= ${cost}`,
            ),
          )
          .returning({ newBalance: goldenNoteBalances.balance });

        if (debited.length === 0) {
          throw new TRPCError({
            code: "BAD_REQUEST",
            message: `Not enough Golden Notes. You need ${cost}.`,
          });
        }
        const newBalance = debited[0].newBalance;

        await tx.insert(userAvatars).values({
          userId: ctx.user.id,
          avatarId: avatar.id,
          acquiredVia: "purchase",
          spentGn: cost,
        });

        await tx.insert(goldenNoteTransactions).values({
          userId: ctx.user.id,
          amount: -cost,
          kind: "spend_avatar_unlock",
          reason: `avatar:${avatar.slug}`,
          balanceAfter: newBalance,
        });

        await tx
          .update(users)
          .set({ equippedAvatarId: avatar.id, updatedAt: new Date() })
          .where(eq(users.id, ctx.user.id));

        return { avatarId: avatar.id, newBalance, equipped: true as const };
      });
    }),
```

```tsx
export type UserAvatarProps = {

  size?: "sm" | "md" | "lg";
  className?: string;

  });

  }

  return (
    <img
      src={resolved}
      alt=""
      className={cn(
        "rounded-full bg-muted object-cover ring-1 ring-border/50",
        SIZE_CLASSES[size],
        className,
      )}
    />
  );
}
```

## Task 12: Render avatars on Leaderboards rows

**Files:**
- Modify: `server/routers/game.ts` (extend `getLeaderboard` return shape)
- Modify: `client/src/pages/Leaderboards.tsx`

- [ ] **Step 1: Extend the leaderboard query to include the equipped slug**

In `server/routers/game.ts`, find the `getLeaderboard` procedure (around line 775). The current implementation does `db.select().from(leaderboardEntries).where(...)` — convert it to a join that pulls the user's equipped avatar slug.

Replace the `.query` body with:

```ts
.query(async ({ input }) => {
  const db = await getDb();
  if (!db) return [];

  const conditions = [];
  if (input.mode) conditions.push(eq(leaderboardEntries.mode, input.mode));
  if (input.genre) conditions.push(eq(leaderboardEntries.genre, input.genre));
  if (input.decade) conditions.push(eq(leaderboardEntries.decade, input.decade));
  if (input.timeframe === "weekly") {
    const weekAgo = new Date(Date.now() - 7 * 24 * 60 * 60 * 1000);
    conditions.push(sql`${leaderboardEntries.createdAt} >= ${weekAgo}`);
  } else if (input.timeframe === "monthly") {
    const monthAgo = new Date(Date.now() - 30 * 24 * 60 * 60 * 1000);
    conditions.push(sql`${leaderboardEntries.createdAt} >= ${monthAgo}`);
  }

  const entries = await db
    .select({
      // Spread the existing leaderboardEntries columns into the response.
      // Using the auto-aliasing form to keep the shape stable for the client.
      entry: leaderboardEntries,
      equippedAvatarSlug: avatars.slug,
    })
    .from(leaderboardEntries)
    .leftJoin(users, eq(leaderboardEntries.userId, users.id))
    .leftJoin(avatars, eq(users.equippedAvatarId, avatars.id))
    .where(conditions.length > 0 ? and(...conditions) : undefined)
    .orderBy(sql`${leaderboardEntries.score} DESC`)
    .limit(input.limit);

  return entries.map((row) => ({
    ...row.entry,
    equippedAvatarSlug: row.equippedAvatarSlug ?? null,
  }));
}),
```

Add the import for `avatars` and `users` if not already:

```ts
import { ..., users, avatars } from "../../drizzle/schema";
```
